## Proposed Algorithm

In this section, we first introduce the overall structure of proposed SCFLNet model, followed by a detailed explanation of the four main modules contained in the model, i.e., CPEM, CAFM, and PCAM, respectively. Finally, we introduce the objective function used for optimizing the model parameters of our newly proposed SCFLNet.

### Overview of Proposed
An overview of the SCFLNet architecture is illustrated in Fig. 2. The proposed SCFLNet focuses on leveraging scale-customized features for accurate crowd counting. The primary task generates the final density map using a multiscale feature extraction network and a series of specialized modules, including CPEM, CAFM, and PCAM. The structure consists of an encoder for feature extraction and a decoder comprising these modules. In the encoder, we use VGG-16 as the network backbone but omit the final pooling layer, all subsequent fully connected layers, and the pooling layers between the fourth and the fifth stages. In the decoder, CPEM employs two branches to enhance features at different scales, ensuring that the feature maps capture both local crowd details and global contextual information. CAFM adaptively fuses the enhanced feature maps with the upsampled feature maps, improving context information integration through deformable convolutions. The fused feature maps from different scales are concatenated along the channel dimension and then passed to PCAM. PCAM integrates the multi-layer outputs of the decoder before upsampling to produce the final density map, enhancing foreground-background separation to ensure high-quality density maps. Additionally, in BCM, masks representing different densities are generated from the ground truth using a mask classifier. By multiplying the enhanced multi-scale features with these masks, the features corresponding to different scales are directed to count individuals of varying densities. This architecture effectively captures and utilizes multi-scale information, allowing the network to adapt to crowds with varying densities and improve overall counting accuracy.

## Algorithm

```Algorithm 1: SCFLNet

Input: Training dataset D = {(I1, D1), ··· , (IN, DN)}
Output: Density map Dpre

1: for i ← 1 to N do
2:     Extract F1, F2, F3 from Ii;
3:     for j ← 1 to 3 do
4:         Feature enhance Fj to FFE-j;
5:     end for

6:     Let F1 = P0;

7:     for j ← 1 to 3 do
8:         Context Feature Fusion (Pj−1, FFE-j) to Pj as Eq.(9);
9:     end for

10:    Channel Attention P to PM as Eq.(11);
11:    Upsample PM to Dpre;
12: end for

13: return Dpre;
```
